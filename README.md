# ColdWatch — IoT Cold Chain Monitoring System

  # COLDWATCH (AN IOT-BASED COLD CHAIN MONITORING SYSTEM FOR PERISHABLE GOODS)

**Live app:** [coldwatch-iot-dashboard.vercel.app](https://coldwatch-iot-dashboard.vercel.app)

## Features

- Real-time temperature and humidity monitoring across multiple devices
- Role-based access — farmers, warehouse managers, transporters
- Alert system with severity levels, auto-resolution, and acknowledgement
- Per-device threshold configuration
- History charts and CSV export
- Celsius / Fahrenheit toggle
- Produce mode presets

## Design

Original Figma design: [IoT Cold Chain Dashboard](https://www.figma.com/design/8gBdLc29zN8gRYOH0IqP7u/IoT-Cold-Chain-Dashboard)
